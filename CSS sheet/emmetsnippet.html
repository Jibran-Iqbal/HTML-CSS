Press Ctrl+Space to get emmet suggestions
In an emmet suggestion we can press read more or info to see what we are going to get in advance This can be helpful when using emmet snippets
After typing an element's name if it is followed by a dot and then a random word then that random word becomes its class name
for example:=
h1.main.second
This will be followed by pressing the tab key
After typing an element's name if it is followed by a hashtag(#) and then a random word then that random word becomes its id name. This seems to be limited to one id only
for example:=
h2#main
This will be followed by pressing the tab key After typing an element's name if it is followed by a hashtag(#) and a then a random word and then a dot and then another random word then that random word becomes its id name and the other one becomes its class name. This seems to be limited to one id only
for example:=
h2.main#main.secondary
This will be followed by pressing the tab key However with a div this is alot easier. All we have to do is type class name or id name or both with dot or hashtag or both and then press tab
for example:=
.main#main.secondary
This will be followed by pressing the tab key We can also create children with div
The syntax will be as follows:=
div>ul>li
This will be followed by pressing the tab key
However this is not limited to just ul and li and it is also not limited to just one list item
We can use multiplication symbol to increase the number of the list items The syntax will be as follows:=
div>ul>li*5
This will be followed by pressing the tab key We can also have text added in our element. The syntax is simply using curly braces
for example:=
p{some text}
div{some text}
span{some text}
This will be followed by pressing tab key
We can also combine it all
for example:=
div>ul>li*6{$}
The dollar sign here is for numbers 1-6
